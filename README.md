# Dark Ascension Labs — Website

Official website for **Dark Ascension Labs**, an indie game studio crafting dark fantasy experiences.

**Live at:** [darkascensionlabs.com](https://darkascensionlabs.com)

## Pages

- **index.html** — Studio homepage with both games, about section, roadmap, community links
- **thelsicker.html** — Dedicated page for Thelsicker Ascends (metroidvania)

## Games

- **Dark Ascension** — Idle RPG available on [iOS](https://apps.apple.com/app/dark-ascension/id6759624526) and [Windows](https://darkascension3000.itch.io/dark-ascension)
- **Thelsicker Ascends** — Dark fantasy metroidvania, coming to Steam, PS5, and Xbox

## Setup

This site is hosted via GitHub Pages. To link the custom domain:

1. In repo Settings > Pages, set source to "Deploy from a branch" (main, root)
2. Add custom domain: `darkascensionlabs.com`
3. Configure DNS at your registrar:
   - A records: `185.199.108.153`, `185.199.109.153`, `185.199.110.153`, `185.199.111.153`
   - CNAME: `www` → `mikejamescalvert.github.io`
4. Enable "Enforce HTTPS"
